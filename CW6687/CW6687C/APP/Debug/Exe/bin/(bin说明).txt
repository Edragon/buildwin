以下顺序即为参数顺序
====== makecode_xx.bin ======
00: 使用P13做升级Key(默认)
01: 使用其他I0做升级Key

====== update_xx.bin ======
00: 不需要SD卡升级
01: 需要SD卡升级

====== table_xx.bin ======
00: 不使用字库转换表

====== font_res_xx.bin ======
00: 不使用字库

====== res_xx.bin ======
00: 不使用图片资源

====== menu_xx.bin ======
00: 不使用菜单图片资源

====== mp3res_xx.bin ======
00: 不使用语音提示功能
01: 使用语音提示功能


====== btinfo_xx.bin ======
00: 蓝牙相关信息
